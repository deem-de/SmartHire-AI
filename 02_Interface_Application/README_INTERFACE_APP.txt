Interface Application Folder
============================

This folder contains the final runnable SmartHire application.

Included parts
--------------
1. Backend API server
2. Resume extraction pipeline
3. Post-processing and Resume Intelligence logic
4. Interview question and evaluation logic
5. Frontend interface
6. Trained models and metrics needed at runtime

Run
---
1. Install dependencies:
   python -m pip install -r requirements.txt
2. Start the application:
   .\start_smarthire.ps1
3. Open:
   http://127.0.0.1:3001
