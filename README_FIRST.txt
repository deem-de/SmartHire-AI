SmartHire - Clean Code Submission
=================================

This package is organized into two main parts only.

01_Model_Pipeline
-----------------
Contains the machine learning pipeline:
- data
- preprocessing
- training
- evaluation
- trained outputs

02_Interface_Application
------------------------
Contains the final application:
- backend server
- resume extraction
- Resume Intelligence logic
- interview logic
- frontend UI
- runtime model files

What was missing in the runtime-only package
--------------------------------------------
The runtime-only package did not include the full model pipeline files such as:
- training
- evaluation
- data loading
- dataset preparation
- final training dataset

This package includes those missing parts.
